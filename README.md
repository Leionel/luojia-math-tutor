## 💡 项目背景 (Pain Points)

目前的通用大模型文采飞扬，但一面对高等数学往往会遭遇“理科翻车”。它们面临三大硬伤：
1. **算不准（计算幻觉）：** 面对高阶矩阵运算和复杂积分，常一本正经地算错数值，误导学生。
2. **画不出（缺乏可视化）：** 诸如方向导数、概率分布等抽象概念，纯文字解释让学生越看越懵，缺乏几何直观。
3. **管不住（暴力剧透）：** 传统搜题 AI 倾向于一次性抛出完整解析，剥夺了学生的思考过程，沦为抄作业工具。

本项目专为被《高等数学》、《线性代数》、《概率论与数理统计》折磨的理工科大学生及考研党设计，提供“权威概念答疑 + 动态图像绘制 + 考研难题分步引导”的导师级服务。

## 🚀 核心架构与创新 (Core Features)

珞珈数智助教采用 **“意图识别 —— 双引擎驱动 —— 多模态输出”** 的复合工作流架构：

### 1. 神经符号双引擎 (Neuro-Symbolic Engine)
我们引入外部 Python “大脑皮层”接管纯粹的数值计算，彻底消灭 AI 幻觉。
* **分支 A（可视化解析）：** 自动调用本地知识库提取权威定义，并由 LLM 编写 `Matplotlib/NumPy` 代码，生成动态几何图（如绘制梯度向量图或切平面）。
* **分支 B（严谨推导引擎）：** 面对复杂习题，由后端编写 `SymPy` 符号计算代码，获取绝对精确的数值解，保证逻辑严密。

### 2. 暗箱自愈风控 (Self-Healing System)
系统后台会强制比对自然语言推理与代码运行结果。若代码报错，AI 会自动触发自我修正机制重写代码并验证，确保最终输出的正确性。

### 3. 自适应助教人格 (Step-by-Step CoT)
引入启发式交互模型。在解题过程中通过提问引导用户思考（如：“你觉得下一步该换元还是分部积分？”），还原真实的助教辅导场景，符合教育心理学。

## 🗄️ 珞珈数学武器库 (Data Moat)

为了支撑高质量的推理引擎，我们构建了包含超 10,000 条高质量结构化记录的三层金字塔数据基座：
* **L1 权威基座：** 从武大同济版现行教材提纯的 500+ 定理与定义。
* **L2 推理引擎：** 通过自研 Python 自动化流水线清洗出的 8,000+ 包含“问题-代码-解答”的三元组数据。
* **L3 评测标准：** 独家整理的 2016-2024 考研数学一真题库。

**自动化提纯流水线：** 原始数据经由专业数学术语矩阵过滤，通过正则算法强制提取代码块并进行 LaTeX 深度脱水，实现全 JSON 结构化物理隔离，确保 100% 匹配考点。
